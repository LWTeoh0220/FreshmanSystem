# 🚀 未來功能與待辦事項清單 (TODO / FUTURE FEATURES)

這裡記錄了我們在先前的開發討論中規劃好、但尚未完全實作的「進階遊戲化與自動化功能」。這些功能將作為後續開發的北極星指標。

## 1. 🤖 跨處室情報自動化管道 (Data Pipeline)
> **目標**：實現真正的「免登入但即時同步」狀態看板。
* **技術架構**：建立自動化爬蟲（Web Scraping），定時抓取學校三大處室的公開資訊。
  * **【教務處公會】**：抓取學期重要學術事件（如：加退選截止日、期中期末考週）。
  * **【學務處後勤】**：抓取宿舍抽籤日程、體檢規定與課外活動即時進度。
  * **【國際處公會】**：抓取最新僑外生公告、居留證延期提醒。
* **執行方式**：使用 GitHub Actions 撰寫 Python 腳本，每日定時爬取並轉為 JSON，前端透過 API 讀取並結合瀏覽器本地端緩存 (LocalStorage)。

## 2. ⚔️ 任務卡關引導機制 (3 種執行任務的方案)
> **目標**：當玩家缺少前置條件或道具時，避免挫折感並維持沉浸體驗。
1. **傳送點指引（快速移動）**：若玩家嘗試解鎖任務卻未達成條件，彈出對話框提供「傳送門」，直接導航至缺少的前置任務節點。
2. **道具提示彈窗（補給站）**：若任務需要特定文件（如：護照影本、證件照），跳出「情報商人」的提示，告知去哪裡可以獲取這些道具（例如去便利商店列印）。
3. **任務分支解鎖（支線機制）**：若主線任務卡住，可提供一個短暫的「支線任務」（例如：先完成體檢表填寫），完成後自動解除主線的封印。

## 5. 🏰 完善 3D 校園沉浸式導覽 (3D Campus Polish)
> **目標**：讓 Three.js / Canvas 實作的 MVP 版本更具 JRPG 遊戲感。
* **卡通渲染風格 (Cel-shading)**：為 3D 場景加上 Toon Shading 與黑色輪廓線 (Outline Effect)，營造動漫或復古遊戲氛圍。
* **光影優化**：調高環境光 (Ambient Light)，賦予空間微暖的米黃色微光，呼應整體的羊皮紙 UI 底色。
* **告示牌系統 (Billboards)**：移除裸露的 3D 浮空文字，改為「懸浮的復古木製告示牌」或「羊皮紙卡片」，並確保始終面向玩家鏡頭。
* **互動提示圈 (Glow Rings)**：在可互動的據點（如教室門口、樓梯口）地面上，加上淡黃色、緩慢旋轉的光圈或魔法陣，引導玩家探索。
